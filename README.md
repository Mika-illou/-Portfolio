# Portfolio

Ce projet est un portfolio personnel moderne, développé en HTML et CSS, permettant de présenter vos compétences, services, réalisations et informations de contact.

## Fonctionnalités
- Page d’accueil avec présentation et réseaux sociaux
- Section À propos
- Liste des services proposés
- Galerie de projets récents
- Formulaire de contact avec carte Google Maps
- Design responsive et animations

## Installation
1. Clonez le dépôt ou copiez les fichiers dans votre dossier local.
2. Ouvrez `index.html` dans votre navigateur.

## Personnalisation
- Modifiez les textes et images dans `index.html` pour refléter votre profil.
- Adaptez les couleurs et styles dans `style.css` selon vos préférences.

## Dépendances
- [Font Awesome](https://fontawesome.com/) pour les icônes
